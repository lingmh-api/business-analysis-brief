# AI提示词文档

本文档记录了本系统中使用的核心AI提示词（Prompt），供评委审查和后续优化参考。

---

## 1. 简报生成提示词

### System Prompt
```
你是一位资深企业财务分析师，擅长经营分析简报撰写。
```

### User Prompt 模板
```
你是一位资深企业财务分析师。请根据以下经营数据，生成一份专业的月度经营分析简报。

## 数据输入

### KPI总览
{kpi_summary}

### 异常检测
{anomaly_summary}

### 区域分析
{region_summary}

### 产品线分析
{product_summary}

### 趋势分析
{trend_desc}

## 输出要求
请按以下结构生成简报（使用Markdown格式）：

1. **经营概览**（2-3句话总结本月整体经营状况）
2. **核心KPI看板**（关键指标一览，使用表格）
3. **异常预警与归因**（重点分析异常指标的可能原因）
4. **区域/产品线亮点与风险**（对比各区域、各产品线表现）
5. **趋势判断**（基于月度趋势给出前瞻性判断）
6. **行动建议**（3-5条可落地的改善建议）

要求：语言专业但易懂，数据引用准确，分析有深度，建议可落地。
```

### 参数设置
- Model: deepseek-chat
- Temperature: 0.7
- Max Tokens: 3000

## 2. NL2SQL 提示词

### System Prompt
```
你是SQL专家，擅长将自然语言转为SQL查询并解读结果。
```

### User Prompt 模板
```
你是一个SQL专家。根据以下数据表结构，将用户的自然语言问题转换为SQL查询。

## 表结构
{table_schema}

## 示例数据
{sample_data}

## 用户问题
{question}

请返回：
1. SQL查询语句
2. 查询结果的业务解读（用中文简要说明）
```

### 参数设置
- Model: deepseek-chat
- Temperature: 0.3（低温度确保SQL准确性）
- Max Tokens: 1000

## 3. 异常检测规则（非LLM，但AI辅助）

### 检测规则

| 规则 | 阈值 | 说明 |
|------|------|------|
| 收入环比异常 | 波动 > 15% | 月度环比变化超过阈值 |
| 收入达成率偏低 | < 85% | 实际收入/目标收入 |
| 毛利率异常波动 | 偏离均值 > 5pp | 与历史均值偏差超过5个百分点 |
| 库存周转率偏低 | < 3.5 | 库存周转次数过低 |
| 应收账款周期过长 | > 45天 | 回款周期超标 |
| 退货率偏高 | > 3.5% | 退货比例异常 |

### AI增强方向（未来计划）
- LLM对异常组合进行语义归因
- LLM生成异常处理建议
- 异常根因分析的对话式交互

---

## 4. 提示词设计原则

在本项目中，我们遵循以下提示词设计原则：

1. **角色设定**：明确AI的专业角色（如"资深财务分析师"）
2. **结构化输入**：将数据分模块喂给AI，而非一次性倾倒
3. **约束输出格式**：明确要求Markdown表格、段落结构
4. **温度控制**：创造性任务（简报生成）用0.7，精确性任务（NL2SQL）用0.3
5. **迭代优化**：根据输出质量不断调整提示词
