## 二、AI工具选型与理由

| 模块 | AI工具 | 选型理由 |
|------|--------|----------|
| 异常检测 | IQR + 同比环比规则 + LLM辅助判断 | 规则快速筛选 + AI语义理解组合 |
| NL2SQL | DeepSeek / 通义千问 API | 中文理解能力强，SQL生成准确率高 |
| 简报生成 | DeepSeek / 通义千问 API | 结构化输出+自然语言洞察生成 |
| 可视化 | Plotly | 交互式图表，适合嵌入报告 |

## 三、核心功能模块

### 3.1 KPI异常检测
- 多维度：区域 × 产品线 × 渠道
- 检测方法：同比/环比波动 > 阈值 → 标记异常
- AI增强：LLM对异常进行语义归因分析

### 3.2 NL2SQL自然语言问数
- 用户输入：自然语言问题
- AI转换：自然语言 → SQL → 执行 → 结果解读
- 示例："华南区产品A上个月毛利率为什么下降？"

### 3.3 简报自动生成
- 模板：经营概览 → KPI指标 → 异常预警 → 归因分析 → 建议
- AI生成：数据结论 → 自然语言段落 → 图表解读

## 四、数据流设计

```
CSV上传 → Pandas清洗 → KPI计算 → 异常标记 → LLM分析 → 简报Markdown → 导出
                    ↓
              NL2SQL引擎 ← 用户自然语言问题
```
